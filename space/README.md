# 论坛前端

`space/` 是校园论坛的 Vue 3 前端项目，提供首页看板、社区动态流、帖子详情、登录注册、通知中心、个人中心、用户主页和管理后台界面。

默认访问地址：

```text
http://localhost:8081
```

后端默认地址：

```text
http://localhost:3000
```

前端运行前请先启动后端，否则登录、发帖、图片、通知等接口无法正常使用。

## 技术栈

- Vue 3
- Vue Router
- Vuex
- Element Plus
- Bootstrap
- jQuery
- Vue CLI

## 页面说明

首页：

- 展示置顶讨论、热度排行榜和热门分区。
- 点击帖子会新开窗口进入帖子详情。
- 点击分区会进入社区并按分区筛选。

社区：

- 展示帖子动态流。
- 支持分区筛选和标题/正文模糊搜索。
- 登录后可以发帖、上传图片、点赞、评论、回复。
- 帖子卡片右上角提供修改、删除、屏蔽用户、屏蔽分区等操作。

帖子详情：

- 展示完整标题、正文、图片、浏览量、点赞数和评论。
- 评论默认展示 10 条，滚动到底部继续加载更多。
- 点击头像或用户名会新开窗口进入用户主页。

通知：

- 展示点赞、评论、删帖等站内提醒。
- 支持单条已读和全部已读。
- 点击通知会新开窗口进入相关帖子详情。

个人中心：

- 展示当前账号、邮箱、头像和头衔。
- 支持修改用户名、头像、密码和账号。
- 展示自己发布过的帖子。
- 管理已屏蔽用户和分区。

用户主页：

- 展示用户头像、用户名、头衔和可查看帖子。
- 账号和邮箱只在访问自己的主页时显示。

管理后台：

- 管理用户、帖子、评论和分区。
- 设置角色、头衔、封禁用户、重置资料。
- 删除帖子、删除评论、置顶帖子。

登录注册：

- 注册需要用户名、账号、邮箱和邮箱验证码。
- 登录使用账号和密码。

## 安装依赖

首次运行前执行：

```powershell
npm install
```

## 启动前端

推荐使用脚本：

```powershell
.\run-space.ps1
```

脚本等价于：

```powershell
npm.cmd run serve -- --host 0.0.0.0 --port 8081
```

启动成功后访问：

```text
http://localhost:8081
```

## 构建

```powershell
npm run build
```

构建产物：

```text
dist/
```

构建时可能出现 Sass legacy API 或 bundle size 警告，这是当前依赖栈的提示，不代表构建失败。只要显示 `DONE Build complete` 即构建成功。

## 开发约定

- 默认后端地址为 `http://localhost:3000`，使用 `VUE_APP_API_BASE_URL` 覆盖。
- 默认前端端口为 `8081`，使用项目根目录 `.env` 中的 `FRONTEND_PORT` 覆盖。
- 使用 `run-space.ps1` 启动时，会根据 `BACKEND_PORT` 自动设置 `VUE_APP_API_BASE_URL`。
- 登录 token 存储在浏览器 `localStorage`。
- 路由使用 hash 模式，例如 `http://localhost:8081/#/space`。
- 帖子详情、用户主页、个人中心和管理后台入口会新开窗口。

## 常见问题

### 页面能打开，但登录失败

检查：

- 后端是否启动在 `http://localhost:3000`。
- 数据库是否启动。
- 管理员账号是否为 `admin`，密码是否与后端首次启动时的 `ADMIN_PASSWORD` 一致。
- 浏览器中是否残留旧 token，可退出登录后重新登录。

### 接口返回 403

常见原因：

- 未登录访问了需要登录的接口。
- 当前账号被禁用。
- JWT 已过期。
- 普通用户访问了管理员页面或接口。

### 头像或帖子图片加载失败

图片由后端 `/uploads/**` 提供。请确认：

- 后端正在运行。
- 图片文件仍在 `backend/uploads/`。
- 前端访问的后端地址是 `http://localhost:3000`。

### 点击帖子或用户主页被浏览器拦截

项目会在部分入口新开窗口。若浏览器拦截弹窗，请允许 `localhost:8081` 打开新窗口。

### 前端端口被占用

项目默认使用 `8081`。可以先回到项目根目录执行：

```powershell
.\stop-all.ps1
```

或手动查找端口：

```powershell
netstat -ano | Select-String ":8081"
```

### npm install 很慢或失败

检查网络和 npm 镜像源。依赖安装成功后再执行：

```powershell
npm run serve -- --host 0.0.0.0 --port 8081
```

### 构建包体积警告

`chunk-vendors` 较大时 Vue CLI 会提示性能警告。当前项目可以正常运行；如果后续需要优化，可再做路由懒加载和组件按需拆分。
