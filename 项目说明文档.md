# 校园论坛项目说明文档

## 1. 项目概述

本项目是一个校园论坛网站，提供面向校园用户的轻量级交流平台。用户可以注册登录、修改个人资料、发布帖子、浏览动态、点赞、评论，并在自己的帖子收到点赞或评论时获得站内提醒。

## 2. 系统组成

| 模块 | 目录 | 说明 |
| --- | --- | --- |
| 后端服务 | `backend/` | Spring Boot + MyBatis Plus，提供账号、资料、帖子、点赞、评论、通知接口。 |
| 论坛前端 | `space/` | Vue 3 + Element Plus，提供论坛页面、登录注册、个人中心、通知中心。 |

## 3. 核心功能

1. 用户注册与登录：后端使用 JWT 做登录状态校验。
2. 个人中心：用户可以修改用户名、头像地址和登录密码。
3. 管理员后台：管理员可以管理用户、帖子和评论。
4. 帖子动态流：用户可以发布帖子，首页和社区页展示帖子列表。
5. 点赞：登录用户可以点赞或取消点赞帖子，同一用户对同一帖子只保留一条点赞记录。
6. 评论：登录用户可以在帖子下发表评论。
7. 站内提醒：当其他用户点赞或评论自己的帖子时，系统创建通知。
8. 通知中心：用户可以查看通知列表、未读角标、单条已读和全部已读。

## 4. 后端接口

### 4.1 账号接口

| 接口 | 方法 | 说明 |
| --- | --- | --- |
| `/user/account/register/` | POST | 注册账号 |
| `/user/account/token/` | POST | 登录并获取 token |
| `/user/account/info/` | GET | 获取当前用户信息 |
| `/user/account/update/` | POST | 修改用户名、头像和密码 |

### 4.2 管理员接口

| 接口 | 方法 | 说明 |
| --- | --- | --- |
| `/admin/users/` | GET | 获取用户列表 |
| `/admin/users/status/` | POST | 启用或禁用用户 |
| `/admin/users/role/` | POST | 设置用户角色 |
| `/admin/posts/` | GET | 获取帖子管理列表 |
| `/admin/posts/delete/` | POST | 删除帖子并清理关联数据 |
| `/admin/comments/` | GET | 获取评论管理列表 |
| `/admin/comments/delete/` | POST | 删除评论 |

### 4.3 帖子接口

| 接口 | 方法 | 说明 |
| --- | --- | --- |
| `/user/post/get/` | GET | 获取帖子列表 |
| `/user/post/write/` | POST | 发布帖子 |
| `/user/post/getComment/` | GET | 获取帖子评论 |
| `/user/post/comment/` | POST | 发表评论 |

### 4.4 点赞接口

| 接口 | 方法 | 说明 |
| --- | --- | --- |
| `/user/agree/add/` | GET | 点赞帖子 |
| `/user/agree/delete/` | GET | 取消点赞 |
| `/user/agree/get/` | GET | 获取用户已点赞帖子 |
| `/user/agree/count/` | GET | 获取帖子点赞数 |

### 4.5 通知接口

| 接口 | 方法 | 说明 |
| --- | --- | --- |
| `/user/notification/list/` | GET | 获取当前用户通知列表 |
| `/user/notification/unread-count/` | GET | 获取未读数量 |
| `/user/notification/read/` | POST | 标记单条通知已读 |
| `/user/notification/readAll/` | POST | 标记全部通知已读 |

## 5. 数据库设计

初始化脚本位于 `backend/sql/forum_schema.sql`。

主要数据表：

- `user`：用户账号、密码和头像信息。
- `post`：帖子内容。
- `comment`：帖子评论。
- `agree`：帖子点赞记录，`pid + id` 唯一。
- `notification`：点赞和评论提醒。

## 6. 启动方式

### 6.1 后端

```powershell
cd backend
.\run-backend.ps1
```

后端端口：`3000`

### 6.2 前端

```powershell
cd space
.\run-space.ps1
```

前端端口：`8081`

管理员后台地址：`http://localhost:8081/#/admin`

默认管理员账号：`admin`。初始密码由后端启动环境变量 `ADMIN_PASSWORD` 决定；未设置时开发默认值为 `admin123456`。

## 7. 测试建议

1. 后端执行 `.\mvnw.cmd test`，确认 Spring Boot 项目可编译。
2. 前端执行 `npm run build`，确认 Vue 项目可构建。
3. 登录后进入“个人中心”，修改用户名和头像地址，确认导航栏头像和名称同步更新。
4. 使用两个账号验证通知流程：用户 A 发帖，用户 B 点赞和评论，用户 A 应收到通知。
